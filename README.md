# OS Performance Dashboard — Backend

The high-performance API engine for the OS Monitoring Dashboard. Built with **Node.js** to fetch and serve real-time system metrics via a lightweight REST API.

✨ Features

* 🚀 **Real-time Telemetry** — Live CPU, memory, disk, and network stats.
* 📊 **System Stats** — Direct hardware load using `systeminformation`.
* ⚡ **Optimized Polling** — Designed for high-frequency requests from React.
* 🌐 **CORS Ready** — Pre-configured for Next.js frontend.

🛠️ Tech Stack

| Technology | Purpose |
| :--- | :--- |
| **Node.js** | Runtime environment |
| **Express.js** | API Framework |
| **Systeminformation** | OS metrics library |

🚀 Getting Started

```bash
# Clone and navigate
git clone [https://github.com/Iamhc/os-performance-backend.git](https://github.com/Iamhc/os-performance-backend.git)
cd os-performance-backend

# Install dependencies
npm install

# Start development server
npm run dev
